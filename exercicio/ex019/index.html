Personalisando Seletores

Vamos aprensentar o uso do seletor (id) determinado por (#) ele é utilizado para determinar um único elemento, e o seletor (class) determinado por (.) com esse podemos agrupar elementos com características semelhantes.

Teste de título

HTML5 é semântico

Para criar um (id), devemos criar dentro da tag (h1) id="nome da variável", deixando claro que isso não alterará o estilo de imediato porque pricisamos utilizar o CSS separadamente.

Aprendendo CSS

Estilos Web

Para criação da (class) utilizamos o mesmo método para a criação do (id), porém como a (class) é para a utilização de vários elementos é recomendado utilizar em conjunto, pois não há sentido criar uma (class) para determinar apenas um elemento.

Para personalisar o id nas CSS precisa identificar por (#) exemplo: #titulo-principal, e com a (class) é parecido muda somente o uso do ponto: .paragrafos-curtos.

Propriedades herdadas

Como vistos anteriormente com as técnicas de estilos: externo, interno, inline que existe uma herárquia a que técnica que prevalece e a inline depois interno e por último a externa. Na configurações personalizadas de (id) e (class) também há nuances.

Descrevendo um exemplo: criamos dois (h1), um com (id) e outro solto, então criamos um seletor (h1) para personalisar a cor por não especificar o (h1) com (id) a cor configurada vai aparecer nos dois (h1). Para colocar uma cor diferente na (h1) com id precisamos criar um seletor com o nome do id e especificar a cor.
