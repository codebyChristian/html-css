Formulários Semântico

Explicação

Na forma semântica, o input sempre vem acompanhado de um label, ele faz a ligação entre o campo nome e a caixa que irá receber o nome. Dentro do label há o parâmetro for que pode receber quanto name ou id do input, neste caso utilizamos o id.

GET vs POST

GET

As informações colocados no formalários ficam visíveis na url.

Quando utilizar:

  • Dados não sensíveis
  • Formulários simples
  • Não ultrapassar 3.000 bytes
  • Apenas texto

POST

As informações ficam escondidas, mas podem ser encontradas pelo DevTools

Quando utilizar:

  • Dados sensíveis
  • Enviar dados acima de 3.000 bytes
  • Enviar outros tipos de arquivos
