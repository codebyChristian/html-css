Principais Listas

Listas Ordenadas

Este tipo de lista a ordem é algo importante, vamos vê um exemplo de listas de serie que mais assisto: Lembrando que o marcador do tipo será o '1', mas há o 'A','a','I' e 'i'.

  1. Ataque dos Titâs
  2. Sout Park
  3. Rick and Morty
  4. Familia da Pesada
  5. Simpsons

Podemos indicar o início da contagem da lista, normalmente pode começa do 1, porém podemos indicar outro inicio para isso vamos usar o parâmetro 'start' que indicarei que iniciará do 2.

A lista ordenada a tag para criaçaõ é 'OL' e fechamento '/OL'.

Lista não Ordenadas

Este tipo de lista não leva em consideração a ordem da lista. Vamos vê um exemplo de lista de itens de casa.

  • Mesa
  • Geladeira
  • Armário
  • Cadeira

Existe três tipos de marcadores o 'disc', 'circle' e o 'square' o exemplo acima é utilizado o 'disc'.

Para lista não ordenada utilizamos a tag 'UL' e para fechamento a tag '/UL'.
