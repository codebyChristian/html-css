A técnica inline

Estilos Locais ou internos é mais vantasojo do que a técnica inline, essa técnica organiza o código na mesma página do HTML (há outra técnica melhor), mas somente pode ser colocada na tag (head) fora dela o estilo local não irá funcionar. Ela é ideal para projetos pequenos, dentro da tag (head) será colocada o nome da tag que será alterada o estilo e depois criar seletore: abrirá chaves e dentro das chaves colocar os parâmetros que as CSS terá.

1ª desvantagem é que apenas o tag de estilos irá toma muito espaço. A 2ª é que caso queira implementar o mesmo estilo em outros páginas terá que usar o Ctrl C e Ctrl V, porque ela é util para apenas um (1) documento.
