Ligações Usando Hiperlink

O link é a essência fundamental do hipertexto, porque permite ligar documentos entre si e permetir a navegação entre páginas.

Âncora

As âncora permite ligações de um ponto ao outro na World Wide Web.

Para criar um hyperlink, devemos criar âncoras através da tag A. O principal atributo dessa tag é o HREF, esta tag coloca o caminho até ela (página) como a URL ou se a página esta no mesmo computador o nome do arquivo.

Há outro atributo chamado HREFLANG que determina de qual idioma é o site, para facilitar o navegador a fazer a tradução automática(Não é obrigatório, mas o buscaderes que não fizer isso são punidos).

Por padrão, sempre que um visitante clica em um hyperlink, o site de destino abre na mesma janela do site que continha esse link. Ou seja, o conteúdo anterior vai deixar de ser exibido para mostrar o novo conteúdo. Mas isso significa se ele clica em um link na página ele não vai voltar a página de origem? Isso pode acontecer, mas há métodos para controlar o usuário, vamos vê alguns:

  • _BLANK: vai abrir o link em uma janela em branco
  • _SELF: vai abrir o link na janela ou frame atual (PADRÃO)
  • _TOP: vai abrir o destino no navegador completo
  • _PARENT: similar ao _TOP, indicar o nome da janela a abrir

Deixando um alerta que o uso de frames é uma técnica quase em desuso, vamos criar exemplos apenas do parâmetros _BLANK e _SELF.

Esse link é seu ou dos outros?

Existe um recurso para links que indica qual é a natureza do destino usando o atributo REL. Ele aceita vários valores:

  • NEXT: o link é para a próxima parte do documento atual
  • PREV: o link vai para a parte anterior do ducumento atual
  • AUTHOR: o link vai para o site do autor do artigo atual
  • EXTERNAL: o link vai para outro site que não faz parte do site
  • EXTERNAL: o link vai para um site não endossado, como um link pago

Esse primeiro link é um link local ou interno, já que ele leva o usuário a outra página dentro do nosso próprio site.

E para fazer Downloads?

Várias vezes já vimos disponível para downloads arquivos de diferentes formatos. Com a atualização para o HTML5 as âncoras receberam atributos especias para isso.

Basta fazer o link diretamente para o arquivo que se deseja efetuar o download e adicionar o atributo download com o valor configurado para o nome do arquivo a ser baixado e o atributo type para indicar ao navegador que tipo de arquivo está sendo baixado. Vamos ver um exemplo:
