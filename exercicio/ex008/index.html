Diferenças das Tags com e sem Semânticas

Frase em Negrito sem Semântica

Nesta frase temos um termo em negrito usando a tag B

Frase em Negrito com Semântica

Já nesta frase temos um termo em destaque usando a tag STRONG.

Apesar de visualmente não existe diferença a semântica dá sentido a ação e isso é uma das bases do HTML 5.

Termo em Itálico Sem Semântica

Nesta frase, temos um termo em itálico usando a tag I.

Frase em Itálico com Semântica

Nesta frase, temos un termo em ênfase usando a tag EM.

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

E mesmo mudando a cor do marcador de um parágrafo ele não interferem em outros sempre por padrão vem a cor amarela.

Caso seja preciso mudar a cor de dezenas ou até centenas de linhas há outra forma de mudar a cor, esse termo é um teste. Lembrando que mudando a cor pela head, a cor das linhas já alteradas não sofreram alteração.

Texto Grande e Texto Pequeno

Estamos criando um texto grande usando a tag BIG que esta obsoleta pois não há semântica e o um texto pequeno usando a tag SMALL que não esta obsoleta neste parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag DEL.

Texto Inserido

Podemos marcar um texto como inserido para dá enfase e indicar que ele foi adicionado e deve ser consedrado pelo leitor, usando a tag INS.

Há o sublinhado com a tag U (que não é semântica)

Texto Sobrescrito

Para inserir fórmulas matemáticas como para elevar um número utiliza a tag SUP: x20+3

Texto Subscrito

Para inserir fórmulas de elementos químicos como para fazer números minísculos, usando a tag SUB: H20
