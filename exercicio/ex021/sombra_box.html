Adicionando sombra

Utiliza a propriedade box-shadow: exemplo: (box-shadow: 3px 5px 4px black), existe uma ordem de precedência: 1º deslocamento horizontal, sombra para o lado direito, valores negativos vai para esquerda. O 2º deslocamento vertical, sombra para baixo, valores negativos vai para cima. 3º O embaçamento, sombra espalha pelo fundo. 4º A cor, é possível ser transparênte.

Bordas Decoradas

As vértices arredondadas serão determinadas assim: (border-radius:10px), desta forma todas as vértices terão arredondamento em 10px, mas poderá ser colocada separadamente cada vértice dessa forma: (border-radius: 10px 20px 30px 40px) e por último pode ser determinado apenas dois valores, que vai agir de forma intercaladas e sempre partindo do canto superior esquerdo, exemplo:(border-radius: 10px 30px;).
