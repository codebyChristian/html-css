Atualmente existe vários formatos de telas com resuluções diferentes, e os sites terão que adaptar-se a todas essas resoluções.

Lembrando que formatos de imagens gigantes iram demorar bastante para abrir em um smartphone antigo e para piorar a imagem irá toma a tela inteira além disso mecanismo de buscas punem sites sites com taxa de retenção baixa e isso é horrível para quem demorou dias ou até semanas para criar um site.

Imagens Flexíveis

Vamos aprender a gerar imagens de tamanho diferentes e fazer o navegador carregar a imagem certa para a resolução certa.

A forma de criar a tag picture e colocar img dentro está certo, porém se a imagem for grande demais ela não caberá no navegador para resolver vamos usar a tag source.

A tag source tem 3 atributos: Type: vai indicar o media type da imagem que usamos. Srcset: vai configurar o nome da imagem que será carregada quando o limite de tamanho for atingida. Media: indica o tamanho máximo a ser considerado para carregar a imagem indicada no atributo srcset.
